Bandapp
Aplicativo de estudos de Lucas Henrique Waki, visando abordar os seguintes temas:
1) a crianção de uma api que alimente um banco de dados com as refeições do bandejão
2) um app de celular que faça notificações com relações a refeições
3) integração com um sistemas de vendas ambulantes
4) integraçaõ de um sistema de analytics básico para os ambulantes
